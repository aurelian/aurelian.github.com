Content Editor is a Beast plugin that allows creation of simple pages with ...stuff.

h3. Features

 * Nice Permalinks (http://example.com/my-cool-page)
 * Admin only (http://example.com/pages to start with it)
 * Textarea with Textile
 * ...umm

h3. Install

Make sure you have a working Beast (or Altered Beast) installation! Then clone with Git:

<pre>git clone git://github.com/aurelian/beast--contenteditor.git vendor/beast/content_editor</pre>

or <a href="http://github.com/aurelian/beast--contenteditor/tarball/master">download</a> and unpack to vendor/beast/content_editor.

There is also a <a href="http://github.com/aurelian/beast--contenteditor/wikis">wiki page</a>.
