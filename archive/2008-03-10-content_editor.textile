<p>Content Editor is a Beast plugin that allows creation of simple pages with &#8230;stuff.</p>



    <h3>Features</h3>



    <ul>

        <li>Nice Permalinks (http://example.com/my-cool-page);</li>

        <li>Admin only (http://example.com/pages to start with it);</li>

        <li>Textarea with Textile;</li>

        <li>&#8230;umm</li>

    </ul>



    <h3>Install</h3>



    <ul>

        <li>make sure you have a working Beast (or Altered Beast) installation!</li>

        <li>clone with Git:



<pre>git clone git://github.com/aurelian/beast--contenteditor.git vendor/beast/content_editor</pre>

</li>

        <li>or <a href="http://github.com/aurelian/beast--contenteditor/tarball/master">download</a> and unpack to vendor/beast/content_editor</li>

        <li>there is a <a href="http://github.com/aurelian/beast--contenteditor/wikis">wiki page</a>.</li>

    </ul>
